## Cache family

**CCM roots**

CCM secular sources, discovery windows, refinements, and spectral windows.

Artifact kinds assigned to this family:

- `ccm_secular_source`
- `ccm_root_discovery_window`
- `ccm_root_refinement`
- `ccm_spectral_window`

## Layout

```text
objects/sha256/<first-two-hex>/<full-sha256>
artifacts/<artifact-kind>/<logical-key>/<parameter-digest>/
```

Object payloads are immutable and content-addressed. Artifact manifests provide
logical identity, provenance, validation state, and ordered object references.
The Xcelerator cache registry is the authority for shard routing.
